TcfGeomPointInPolygon
Form TcfGeomPointInPolygon
Description Uses a ray casting approach to count if the number of intersections is even or odd. Even means the point is outside, and odd means inside. Returns t if inside the polygon. Note that the algorithm will ensure that for abutting polygons, it is only inside one of them. So, for a single polygon, needs to be combined with TcfGeomPointOnBoundary to know if on boundary or inside. http://en.wikipedia.org/wiki/Point_in_polygon
See also Skill/Skill++ source file TcfGeomPointInPolygon
