TcfXml
Form TcfXml (@rest addTags)
Description Main function which operates over the TcfXmlObj data structure. Better support for XML (both HTML and XML are handled with the same API). (can specify whatever tags you like when you call TcfXml - e.g. TcfXml("tag1" "tag2" "tag3")). Added a ->start_xml method. Also supports tags with no content.
Parameters addTags variable number of XML entry tags
Examples 
Example: 
xml=TcfXml("ROOT" "DATA" "INFO") 
xmlFile=outfile("out.xml") 
fprintf(xmlFile "%s" xml->start_xml( obj )) 
fprintf(xmlFile "%s\n" 
 xml->ROOT( 
  ?hello "world" 
  ?attribute "value" 
  xml->DATA( 
  xml->INFO(?myAttrib "val1") 
  ) 
  ) 
  ) 
close(xmlFile)
