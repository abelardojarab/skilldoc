TcfFloorplanCluster
Form defun TcfFloorplanCluster ( obj )
Description Defines a class for the cluster structure (cluster instance is different than cluster structure). Use proper class notation for UML 2.0. Greater flexibility than a DPL as properties are automatically initialized when using a class.
Parameters ClusterName name of the cluster
ClusterLibrary library name for the cluster
isSeparateCv if set to true ('t'), then the cluster is built as a separate cell with name ClusterName on libray ClusterLibrary
isRoot if set to true ('t'), the cluster is the root cell of the hierarchy tree
isLeaf if set to true ('t'), the cluster is bottom of the hierarchy tree (e.g. a hand-drawn or unit layout)
isBaseACell if set to true ('t'), the base of the cluster is a cell (thus the base of the cluster can be placed as an instance in the cluster)
Examples 
(foreachs cellstring celllist
   (setq IndDPL (list nil))
   (setq IndividualCell (parseString cellstring "."))
   (putprop IndDPL (nth 0 IndividualCell) 'LibraryName)
   (putprop IndDPL (nth 1 IndividualCell) 'CellName)
   (putprop IndDPL (nth 2 IndividualCell) 'Rotation)
   (putprop IndDPL (nth 3 IndividualCell) 'alignHandle)
   (putprop IndDPL (quotient (atof (nth 4 IndividualCell)) 1000) 'xSep)
   (putprop IndDPL (quotient (atof (nth 5 IndividualCell)) 1000) 'ySep)
   (putprop IndDPL (nth 6 IndividualCell) 'refObj)
   (putprop IndDPL (nth 7 IndividualCell) 'refHandle)
   (setq C (makeInstance 'TcfFloorplanCluster
          ?ClusterName (symbolToString (concat IndDPL->CellName iteration))
          ?isBaseACell nil
          ?isSeparateCv nil
          ?isLeaf t
          ?isRoot nil
  ))
   (TcfFloorplanClusterSetBase C IndDPL->CellName IndDPL->LibraryName nil IndDPL->xSep IndDPL->ySep IndDPL->Rotation)
   (setq I (makeInstance 'TcfClusterInstance ?RefName IndDPL->refObj ?RefOffsetX IndDPL->xSep ?RefOffsetY IndDPL->ySep ?RefAlignHandle IndDPL->alignHandle ?RefRefHandle IndDPL->refHandle ?isRefPlaced t ?isPlaced nil ?Cluster C))
   (TcfFloorplanAddClusterToCluster F I)
   (setq iteration iteration+1)
  ) ;; foreach
